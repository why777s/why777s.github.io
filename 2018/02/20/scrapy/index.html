一. 配置

1. 安装(macOS)

先安装c++库,再安装scrapy

xcode-select –install
pip3 install scrapy

2. 创建一个项目

scrapy startproject ProjectName

3. 创建一个spider

参数为spider的名称及爬取的域名

scarpy genspider quotes quotes.toscrape.com




二. 提取数据

爬取的目标站点是http://quotes.toscrape.com/

1. 进入Shell

scrapy shell ‘http://quotes.toscrape.com/‘

2. 使用response对象的css选择器

返回一个SelectorList对象，可以通过extract()或extract_first()提取内容（相当于一个是find_all一个是find()）
‘title::text’意味着选择在title标签中的text元素；通过extract()提取title中的内容
