三. 命令行工具

  1. startproject:新建Scrapy项目
    scrapy startproject <project_name> [project_dir]
  2. genspider:在当前目录或当前项目的目录下创建一个爬虫
    scrapy genspider <name> <domain>
  3. crawl:开始爬取
    scrapy crawl <spider>
  4. list:列举当前项目中所有可用的爬虫
    scrapy list
  5. fetch:下载指定网页并返回内容
    scrapy fetch <url>
    可选参数有:--headers:打印头、--nolog:不打印日志等
    e.g
    scrapy fetch --nolog --headers http://www.baidu.com
  6. view:在浏览器中打开指定URL
    scrapy view <url>
  7. shell：启动Scrapy终端
    scrapy shell [url]
  8. version:显示scrapy版，加上-v参数显示更多
    scrapy version [-v]
  9. bench:运行一个测试，测一下速度什么的
    scrapy bench

2. 几个基本的属性和方法：

  1. name: Spider的名字，唯一标识，必须的
  2. allowed_domains: 包含了Spider允许爬取的域名列表，可选。
  3. start_urls: URL列表。当没有指定url时，Spider从该列表中爬取。
  4. parse(response）: 负责处理response并返回处理的数据以及/或跟进的URL。请求连接后默认调用。

五. 选择器Selectors

进入shell爬取官方文档中提供的实例网站
scrapy shell https://doc.scrapy.org/en/latest/_static/selectors-sample1.html

1. 基本用法

  1. Scrapy选择器通过特定的XPath或者CSS表达式来选择HTML文档中的内容。
  2. 在Response中使用XPath和CSS查询非常普遍，故而提供了两个快捷方式，他们都返回一个SelectorList（内建List的子类）的实例
    • response.xpath()
    • response.css()
  3. 使用.extract()和.extract_first()获取原文内容（前者提取所有，后者提取匹配到的第一个，若无匹配返回None）
  4. 在标签后加上::text选择文本内容
  5. 配合正则表达式: Selector也有.re()和.re_first()方法
    e.g 爬取图片的名字
