一. Redis安装

Mac

  1. 安装Homebrew（使用Homebrew安装redis最方便）
    将以下命令粘贴至终端

    /usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”

  2. 安装redis

    brew install redis

  3. 使用redis服务

    redis-server /usr/local/etc/redis.conf

  4. 启动redis

    redis-cli

  5. 检测redis服务是否启动

    127.0.0.1:6379> ping
    PONG

  • string
  • hash：一个键值对的集合
